Project Requirements

Python

Version: Python 3.x (Python 3.6 or later is recommended)
Download: Python Official Site

IDE or Text Editor

Memory: At least 2 GB of RAM (sufficient for running Python scripts and development tools)
Disk Space: Minimal; Python and the required libraries do not require significant disk space
